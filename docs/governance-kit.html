FAQ

What format are the templates?
Markdown (.md) — works in any editor, Notion, Confluence, GitHub, VS Code. Easy to convert to Google Docs, Word, or PDF.
How long does implementation take?
Quick start: 1 hour (inventory your agents, classify them, present to leadership). Full implementation: 1 week (customize policy, complete risk assessments, set up IR playbook, map compliance).
Do I need Paybound to use this?
No. The governance kit defines WHAT controls you need. Paybound is one tool that implements them, but the templates work with any stack.
Is this relevant if we only have a few agents?
Especially then. Starting with governance when you have 2-3 agents is 10x easier than retrofitting it when you have 50. The templates scale down as well as up.
What's the difference between this and the Security Playbook?
The Security Playbook ($29) teaches the technical security layers for individual agents. The Governance Kit ($79) provides the organizational framework — policies, processes, compliance mapping — for managing agents across your company. They complement each other.
